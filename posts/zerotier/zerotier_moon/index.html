Prerequisites:

  • A server with pubilc IP addr

  • Zerotier installed

  • Joined your network

  • Server is authorized

On the Server#

1

cd /var/lib/zerotier-one

the following commands might require root

1

zerotier-idtool initmoon identity.public > moon.json

modify stableEndpoints in moon.json

1

"stableEndpoints": ["<your server IP>/9993"]

1

zerotier-idtool genmoon moon.json

then a file similar to 000000xxxx.moon will be generated

1

mkdir moons.d

1

mv 000000xxxx.moon moons.d

1

sudo systemctl restart zerotier-one

run zerotier-cli peers to check whether you server is Moon now

On the Windows#

go to C:\ProgramData\ZeroTier\One

create dir moons.d

copy the 000000xxxx.moon generated in you server to above dir

exit Zerotier client

go to service.msc to restart the zerotier service

launch the Zerotier client

run cmd as Administrator

check Moon

1

zerotier-cli peers

On the NAS#

copy 000000xxxx.moon to a dir in your NAS

1

docker exec -it zerotier mkdir /var/lib/zerotier-one/moons.d

1

docker cp <moon file dir> <container ID>:/var/lib/zerotier-one/moons.d

1

docker exec -it zerotier zerotier-cli orbit <your Moon node Zerotier ID> <your Moon node Zerotier ID>

check Moon

1

docker exec -it zerotier zerotier-cli peers
